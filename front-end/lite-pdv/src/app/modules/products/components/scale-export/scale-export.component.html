Exportação para balança

Informações adicionais

O arquivo gerado é compatível com as balanças Toledo modelo Prix Uno 4.

  • As balanças devem pertencer ao departamento 01;
  • Imprimir etiquetas com o código de barras no padrão EAN-13 composto da seguinte forma:
  • 2 - Produtos in-store (sempre será 2);
  • C - Código do produto com 6 dígitos;
  • P - Peso do produto;
  • D - Dígito verificador do código de barras;
